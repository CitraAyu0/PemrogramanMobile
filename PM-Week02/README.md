Pada pertemuan ini saya mempelajari dasar-dasar bahasa pemrograman Dart yang digunakan dalam pengembangan aplikasi mobile. Materi dimulai dari pengenalan Dart sebagai bahasa yang berorientasi objek dan memiliki fitur null safety. Setiap program Dart dijalankan melalui fungsi main() sebagai titik awal eksekusi.

Selanjutnya dipelajari tentang variabel sebagai tempat menyimpan data di dalam program. Variabel dapat ditulis menggunakan var atau dengan tipe data secara langsung seperti int, double, String, dan bool. Selain itu, ada juga penggunaan final dan const untuk variabel yang nilainya tidak berubah.

Materi juga membahas tipe data primitif dan non-primitif seperti List dan Map yang digunakan untuk menyimpan banyak data. Kemudian dipelajari berbagai operator, seperti operator aritmatika untuk perhitungan, operator perbandingan untuk membandingkan nilai, operator logika untuk kondisi, serta operator penugasan dan null-aware.

Dari materi ini saya memahami bahwa variabel, tipe data, dan operator merupakan dasar penting dalam menyusun logika program. Pemahaman konsep ini sangat diperlukan sebelum membuat aplikasi yang lebih kompleks.
