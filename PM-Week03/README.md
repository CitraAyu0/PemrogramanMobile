Nama : Citra Ayu Meilinda

Pada praktikum hari ini, saya mempelajari bagaimana menggunakan struktur kontrol dalam bahasa Dart, khususnya percabangan (if-else) dan perulangan (while). Melalui pembuatan program Game Tebak Angka, saya memahami bagaimana logika program bekerja untuk membandingkan input pengguna dengan angka rahasia, memberikan petunjuk, serta mengulang proses hingga kondisi tertentu terpenuhi. Saya juga belajar pentingnya validasi input menggunakan int.tryParse agar program tidak mengalami error ketika pengguna memasukkan data yang tidak sesuai.

Selain itu, saya mempelajari penggunaan library seperti dart:io untuk menerima input dari pengguna dan dart:math untuk menghasilkan angka acak. Dari sini saya memahami bahwa sebuah program sederhana tetap membutuhkan pengelolaan alur yang terstruktur agar berjalan dengan baik. Konsep variabel, perulangan bersarang (nested loop), dan penghitungan jumlah percobaan juga membantu saya memahami bagaimana mengatur logika yang lebih kompleks secara bertahap.

Tidak kalah penting, pada praktikum ini saya juga belajar menggunakan Git sebagai version control system. Saya memahami fungsi commit untuk menyimpan perkembangan kode, branch untuk mengembangkan fitur secara terpisah, serta merge untuk menggabungkan perubahan ke branch utama. Dengan menggunakan beberapa branch dan commit, saya dapat melihat proses pengembangan program secara sistematis dan terstruktur. Hal ini sangat membantu dalam pengelolaan proyek pemrograman agar lebih rapi dan profesional.
